# Risbow Launch Site

A Next.js + Tailwind CSS launch page showcasing Razorpay payment compliance and SMS provider readiness.

## Getting Started

```bash
# install dependencies
npm install

# run local dev server
npm run dev

# create production build
npm run build

# run production preview
npm run start
```

- Dev server: http://localhost:3000
- Primary content: pages/index.tsx
- Tailwind utilities: styles/globals.css and tailwind.config.ts
- Launch blueprint includes hero, compliance cards, timeline, FAQ, and package summary sections.
